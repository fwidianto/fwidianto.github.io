HS Code Trade Compliance Automation

Overview
--------
Automates retrieval of Indonesian import tariff and licensing
requirements from the INSW portal.

Features
--------
- Browser automation using Playwright
- Automatic detection of missing HS records
- Tariff extraction
- Permit extraction
- Excel export
- JSON export

Technology
----------
- Python
- Playwright
- OpenPyXL
- Regex

Output
------
Sheet: HS Code Tarif
Sheet: HS Code Izin
JSON: hasil_hs.json
