📡 Web Scraping Automation Project

Portfolio Module

PROJECT OVERVIEW

Automated scraping system that extracts HS Code import document data and structures it into Excel for analysis and reporting. Built to eliminate manual data entry and improve consistency in trade data workflows.

IMPACT

-80% Time

Manual processing reduced significantly through automation.

DATA SOURCE

Web + Excel

Batch HS Code extraction with structured parsing.

AUTOMATION

Python

Scheduled scraping + Excel pipeline.

TECH STACK

  • Python (automation & scraping)
  • BeautifulSoup / Requests
  • OpenPyXL (Excel processing)
  • Google Sheets API (optional sync)
  • Windows Task Scheduler

WORKFLOW

  • Load HS Code list from Excel
  • Detect missing rows automatically
  • Scrape data from target web source
  • Clean + normalize dataset
  • Write back into structured Excel sheets

CHALLENGE & SOLUTION

The main challenge was inconsistent HTML structure and misaligned Excel indexing. This was solved by implementing dynamic row mapping and validation checks before writing data.
